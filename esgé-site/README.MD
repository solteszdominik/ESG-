# Seres – Száraztészta Manufaktúra Weboldal

Ez a projekt egy magyar kézműves száraztészta-manufaktúra bemutatkozó weboldala.  
Célja egy letisztult, modern és reszponzív frontend felület megvalósítása, amely egyszerre informatív és vizuálisan igényes. A későbbiekben fejlesztésre kerül egy webshop.

> ⚠️ A projekt jelenleg fejlesztés alatt áll, folyamatosan bővül új funkciókkal és tartalommal.

---

## Projekt célja

- A márka történetének és termékeinek bemutatása
- Felhasználóbarát, reszponzív weboldal készítése
- Frontend technológiák gyakorlása
- Portfólió projektként való felhasználás
