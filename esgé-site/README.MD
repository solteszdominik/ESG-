# Seres – Pasta Manufactory Website

This project is a presentation website for a Hungarian artisanal dry pasta manufactory.  
Its goal is to create a clean, modern, and responsive frontend interface that is both informative and visually appealing. In later phases, the project will be extended with an online webshop.

> ⚠️ The project is currently under development and is continuously expanding with new features and content.

---

## Project Goals

- Present the brand story and its products
- Create a user-friendly, responsive website
- Practice and improve frontend development skills
- Use the project as a portfolio reference
